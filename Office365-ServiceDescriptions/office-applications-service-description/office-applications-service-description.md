# <a name="office-applications-service-description"></a>Office 应用程序服务说明

Microsoft 365是一项订阅服务，为你提供你已经熟悉的Office桌面应用程序的最新版本，例如 Word、Excel 和PowerPoint。
  
即使Office是基于云的服务，Office应用程序也不会在云中运行。 但是，用户可以从 Office 365 门户下载 Office 应用程序并在其本地计算机上安装它们。 Microsoft 使用即点即用技术，让 Office 应用程序的下载和安装既简单又迅速。 即点即用使用虚拟化技术在本地计算机上的一个独立环境中运行 Office 应用程序，可让用户同时运行 Office 应用程序和较早版本的 Office。
  
Office 应用程序有 32 位和 64 位版本。
  
有关为用户启用Office应用程序服务说明的订阅的详细计划信息，请参阅[完整的订阅比较表](https://go.microsoft.com/fwlink/?linkid=2139145)。
