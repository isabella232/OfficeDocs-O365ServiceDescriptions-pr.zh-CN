# <a name="excel-for-the-web"></a>Excel 网页版

Excel Web (以前Excel Web App) 将 Microsoft Excel 体验扩展到 Web 浏览器，您可以在 Web 浏览器中直接在存储工作簿的网站上处理工作簿。 所有客户都可以使用 Web Office查看Office编辑文件。
  
> [!NOTE]
> 超过 50 MB (MB) 无法从 Excel 内查看 Web Microsoft Office SharePoint Online。 要查看较大的文件，需要在您的计算机上安装 Excel 桌面应用程序。 

> [!NOTE]
> 为了帮助你比较产品/服务，本文中列出的高级功能包括 Excel 桌面应用功能，这些功能仅在 Office 套件 (中可用，如 Office Professional Plus、Office Standard 或 Microsoft 365 企业应用版) 。 有关 Web 功能的所有Office（如 Word 网页PowerPoint）的列表，请参阅 Web 计划的跨 Office[的功能可用性](office-online-service-description.md#feature-availability-across-office-for-the-web-plans)。

若要了解 Web 支持哪些文件类型Excel，请参阅支持的 web Office[文件类型](office-online-service-description.md#supported-file-types-for-office-for-the-web)。 若要详细了解 Web Excel 和 Excel 桌面应用之间的差异，请参阅浏览器和 Excel 中的工作簿[Excel。](https://support.office.com/article/f0dc28ed-b85d-4e1d-be6d-5878005db3b6)
