# <a name="powerpoint-for-the-web"></a>PowerPoint 网页版

PowerPoint 网页 (以前PowerPoint Web App) 将 Microsoft PowerPoint 体验扩展到 Web 浏览器，可以在存储演示文稿的网站上直接处理演示文稿。 使用 Office 网页版的 Microsoft 365 客户可以一起查看、创建和编辑文件。
  
PowerPoint 编辑器是一个 Web 前端组件，可创建基于浏览器的编辑图面，使用户可以处理文档而不会丢失保真度。
  
如果选择 PowerPoint 网页工具栏上的" **在 PowerPoint** 中打开"按钮，演示文稿将在 PowerPoint 桌面应用程序中打开 (如果Microsoft PowerPoint 2010或更高版本安装在 powerPoint 桌面) 。
  
了解如何在[电脑上使用 Microsoft 365 商业版下载和安装 Office。](https://support.office.com/article/72977511-dfd1-4d8b-856f-405cfb76839c)
  
## <a name="alignment-bullets-numbers"></a>Alignment, bullets, numbers

借助 PowerPoint 网页，可以更改段落对齐方式、应用项目符号或编号，以及更改项目符号或编号文本的级别。 详细了解[在浏览器与 PowerPoint 桌面应用程序中使用演示文稿的区别](https://go.microsoft.com/fwlink/?LinkId=272763)。
  
## <a name="apply-basic-transitions-and-animations"></a>Apply basic transitions and animations

从动画和过渡效果库中选择。 PowerPoint 网页版支持八种切换和 37 种动画。 PowerPoint 不支持的其他动画和切换在演示文稿中保留，并显示在幻灯片放映中，但无法修改在 PowerPoint 网页中。 详细了解[在浏览器与 PowerPoint 桌面应用程序中使用演示文稿的区别](https://go.microsoft.com/fwlink/?LinkId=272763)。
  
## <a name="apply-rich-formatting-to-text-shapes-and-pictures"></a>Apply rich formatting to text, shapes, and pictures

在 PowerPoint 网页中不可用。 PowerPoint 桌面应用程序中提供了高级格式设置功能，如更多颜色、渐变、取色器、效果和样式。 通过 PowerPoint 网页，可以添加文本框或从形状库中选择并应用定义填充、轮廓和阴影效果的样式。 
  
## <a name="apply-themes-and-theme-variants"></a>应用主题和主题变量

向演示文稿添加新幻灯片时，请从内置主题和变体库中进行选择。 不受 PowerPoint 网页支持的主题保留在演示文稿中，但在 PowerPoint 网页中无法修改主题。 详细了解[在浏览器与 PowerPoint 桌面应用程序中使用演示文稿的区别](https://go.microsoft.com/fwlink/?LinkId=272763)。
  
## <a name="office-add-ins"></a>Office 外接程序

PowerPoint 网页仅支持 Office 内容加载项。 内容外接程序将基于 Web 的功能集成为可以与演示文稿一起显示的内容。 详细了解 [Office 外接程序类型](/previous-versions/office/office-2013-resource-kit/jj219429(v=office.15))。
  
## <a name="arrange-objects"></a>排列对象

在 PowerPoint 网页版中，在图层中从后向前移动、调整大小、旋转形状和文本框或对形状和文本框排序。 对形状取消分组，以对其进行单独处理。 详细了解[在浏览器与 PowerPoint 桌面应用程序中使用演示文稿的区别](https://go.microsoft.com/fwlink/?LinkId=272763)。
  
## <a name="broadcast-slide-show"></a>Broadcast slide show

需要 PowerPoint 桌面应用程序才能通过 PowerPoint 网页版向远程受众广播幻灯片放映。 无论是否安装了 PowerPoint 桌面应用程序，观看者都可以通过 PowerPoint 网页版观看直播演示文稿。 详细了解[广播幻灯片放映](https://go.microsoft.com/fwlink/?LinkId=272765)。
  
## <a name="clipboard"></a>剪贴板

使用 PowerPoint 网页版，可以在演示文稿中剪切、复制和粘贴内容。 用户可以在 Office 网页版程序之间以及 Office 网页版和计算机上桌面Microsoft Office复制和粘贴文本。 详细了解[在浏览器与 PowerPoint 桌面应用程序中使用演示文稿的区别](https://go.microsoft.com/fwlink/?LinkId=272763)。
  
## <a name="create-and-manage-slides"></a>Create and manage slides

在 PowerPoint 网页中添加、重新排序、复制、隐藏和删除幻灯片。 详细了解[在浏览器与 PowerPoint 桌面应用程序中使用演示文稿的区别](https://go.microsoft.com/fwlink/?LinkId=272763)。
  
## <a name="create-custom-animation"></a>Create custom animation

PowerPoint 网页包含动画效果库。 PowerPoint 不支持的 Web 动画保留在演示文稿中，并显示在幻灯片放映中，但不能在 PowerPoint 网页中进行修改。 必须在计算机上安装 PowerPoint 桌面应用程序，才能创建自定义动画。 详细了解如何[使用 PowerPoint 2013 桌面应用程序创建自定义动画](https://go.microsoft.com/fwlink/p/?LinkId=282348)。
  
## <a name="design-tools-advanced"></a>设计工具，高级

在 PowerPoint 网页中不可用。 仅 PowerPoint 桌面应用程序提供了高级设计功能，如创建幻灯片母版和修改版式。 借助 PowerPoint 网页版，可以使用自己的模板文件作为创建新文件的基础，也可以从预定义的主题列表进行选择。
  
## <a name="dropbox"></a>Dropbox

Dropbox 是托管提供云存储、文件同步、个人云和客户端软件的服务的文件。
  
## <a name="embed-presentation-on-web-or-blog-page"></a>在 Web 或博客页面上嵌入演示文稿

借助 PowerPoint 网页版，可以在网站或博客上嵌入演示文稿，以便任何人都可以查看你的信息，即使他们没有 Microsoft PowerPoint 桌面应用程序。 嵌入式查看器可显示动画、切换效果和音频/视频，与全保真阅读视图相同。 详细了解如何[在网站或博客页面上嵌入演示文稿](https://go.microsoft.com/fwlink/?LinkId=272766)。
  
## <a name="font-formatting"></a>字体格式

PowerPoint 网页应用可使你向文本应用粗体、斜体、下划线、字体、大小和颜色。 您还可以使用格式刷复制整个形状的格式。
  
## <a name="full-fidelity-reading-view"></a>全保真阅读视图

按预期方式查看演示文稿，其中显示动画、切换 &mdash; 和音频/视频。
  
## <a name="full-ink-support"></a>Full ink support

墨迹无法插入到 PowerPoint 网页中，但 PowerPoint 网页 PowerPoint 会按预期显示墨迹。
  
## <a name="full-selection-of-animations-and-transitions"></a>Full selection of animations and transitions

PowerPoint 网页版支持八种切换和 37 种动画。 如需更多选择，请使用 PowerPoint 桌面应用程序。 
  
## <a name="headers-and-footers"></a>页眉和页脚

页眉和页脚（包括日期和幻灯片编号）在 PowerPoint 网页中无法插入、编辑或删除，但 PowerPoint 网页 PowerPoint 会按预期显示页眉和页脚。 详细了解[在浏览器与 PowerPoint 桌面应用程序中使用演示文稿的区别](https://go.microsoft.com/fwlink/?LinkId=272763)。
  
## <a name="hyperlinks"></a>Hyperlinks

插入、编辑和跟踪超链接。 书签链接可以正常工作，但不能在 PowerPoint 网页中编辑。
  
## <a name="insert-online-video"></a>插入联机视频

视频和音频内容在阅读视图中和幻灯片放映模式下播放，文件大小限制为 100 MB。 可以从 YouTube 插入联机视频，可以在 PowerPoint 网页版中调整媒体控件的大小、移动和删除媒体控件。 但是，若要插入除 YouTube 外的其他音频和视频，您需要 Microsoft PowerPoint 桌面应用程序。
  
## <a name="integration-with-excel-for-charts"></a>Integration with Excel for charts

在 PowerPoint 网页中不可用。 借助 PowerPoint 网页，可以在现有演示文稿中查看 Excel 图表，但不能使用 PowerPoint 网页视图在演示文稿中编辑或插入 Excel 图表。 详细了解如何[将 Excel 图表复制到 PowerPoint 中](https://go.microsoft.com/fwlink/?LinkId=272773)。
  
## <a name="navigation---slide-sorter"></a>导航 - 幻灯片浏览

幻灯片浏览视图，它为你提供缩略图形式的幻灯片视图，便于对幻灯片进行排序和组织。 可以使用幻灯片浏览视图来组织幻灯片、添加节以及将幻灯片分为不同的类别。
  
## <a name="offline-viewing-and-authoring"></a>Offline viewing and authoring

PowerPoint 网页版本从 Web 浏览器启动，并依赖 Internet 连接。 要脱机访问演示文稿，必须在计算机上安装 Microsoft PowerPoint 桌面应用程序，用于查看和编辑幻灯片。
  
## <a name="picture-cropping"></a>图片裁剪

使用裁剪工具提高图片的主题帧。只需单击图片边缘的其中一个裁剪控点并拖动，直到获得所需的图片。
  
## <a name="pictures"></a>图片

借助 PowerPoint 网页，可以插入存储在计算机中的图片，也可以插入必应图像中的图片。 可以移动、调整和裁剪图片，并应用许多图片样式。 用于处理图片的更复杂的功能（如应用效果）在 PowerPoint 网页中不可用。 在 PowerPoint 网页中无法创建屏幕截图，但演示文稿中的屏幕截图在 PowerPoint 网页图片中显示为图片。 详细了解[在浏览器与 PowerPoint 桌面应用程序中使用演示文稿的区别](https://go.microsoft.com/fwlink/?LinkId=272763)。
  
## <a name="present-online-through-skype-for-business-or-the-office-presentation-service"></a>Present online through Skype for Business or the Office Presentation Service

在 PowerPoint 网页中不可用。 Microsoft PowerPoint 桌面应用程序允许您使用 Office Presentation Service 传递演示文稿，这是一项免费的公共服务，允许其他人在 Web 浏览器中继续操作。 详细了解 [Office Presentation Service](https://go.microsoft.com/fwlink/?LinkId=272774)。
  
## <a name="presenter-view"></a>Presenter view

在 PowerPoint 网页中不可用。 演示者仅可通过 Microsoft PowerPoint 桌面应用程序对演示文稿流、备注、批注和缩放工具进行幕后控制。 PowerPoint 网页不支持这些功能。
  
## <a name="print-to-pdf"></a>打印为 PDF 格式

借助 PowerPoint 网页版，您可以将演示文稿打印到 PDF 阅读器，其中幻灯片的所有版式和格式都将以预期方式打印。 详细了解使用 [PowerPoint 网页时可以执行的基本任务](https://go.microsoft.com/fwlink/?LinkId=272775)。
  
## <a name="proofing-tools"></a>校对工具

可以使用 PowerPoint 网页版中的内置词典检查拼写和设置校对语言。 但 PowerPoint 网页版不使用自定义词典，也不包括翻译或同义词库。
  
## <a name="real-time-co-authoring"></a>实时共同创作

多个作者可以在 PowerPoint 网页版和 PowerPoint 2016 中同时工作。 实时状态可帮助您查看共同创作者在演示文稿中的编辑位置，从而不会在编辑时产生冲突，并且您可以在发生更改时进行查看。 PowerPoint 2013 支持同时编辑，但没有状态指示，并且在保存文档之前多个作者无法看到更改。 有关 PowerPoint 中的实时共同创作详细信息，请参阅在 [PowerPoint 演示文稿上协同工作](https://go.microsoft.com/fwlink/?linkid=844014)。
  
## <a name="reviewer-tools-advanced"></a>审阅者工具，高级

在 PowerPoint 网页中不可用。 仅 PowerPoint 桌面应用程序中提供了高级审阅功能，如合并冲突和比较演示文稿。 借助 PowerPoint 网页，可以查看、添加、编辑或删除注释。
  
## <a name="rights-management-apply-and-consume-irm-and-password-protection"></a>权限管理：应用和使用 IRM 和密码保护

在 PowerPoint 网页中不可用。 PowerPoint 网页显示受信息权限管理或 IRM (保护) 。 但是，这些演示文稿在浏览器中无法编辑，并且你无法为 Web 在 PowerPoint 中创建受 IRM 保护的演示文稿。 PowerPoint 网页应用无法打开使用密码进行数字签名或加密的演示文稿。 
  
## <a name="run-slide-show"></a>Run slide show

借助 PowerPoint 网页，只需使用 Web 浏览器和 Internet 连接，就可以运行演示文稿。 详细了解使用 [PowerPoint 网页时可以执行的基本任务](https://go.microsoft.com/fwlink/?LinkId=272775)。
  
## <a name="save-as-or-download-a-copy"></a>另存为或下载副本

每次更改时，PowerPoint 网页都会保存你的工作。 没有"保存 **"** 命令。 可以下载副本，但必须具有 Microsoft PowerPoint 桌面应用程序才能编辑本地副本。
  
## <a name="shapes"></a>形状

艺术字和图表无法插入到 PowerPoint 网页中，但 PowerPoint 网页 PowerPoint 按预期显示它们。 在"编辑"视图中，编辑文本并设置其格式。 如果要将文本效果应用于艺术字，则需要 Microsoft PowerPoint 桌面应用程序。
  
## <a name="share"></a>共享

如果将演示文稿保存在 SharePoint 文档库中，演示文稿会处于联机状态，可以通过发送链接（而非电子邮件附件）进行共享。拥有适当权限的人员可以在其 Web 浏览器或移动设备上查看演示文稿。详细了解如何[共享演示文稿](https://go.microsoft.com/fwlink/?LinkId=272776)。
  
## <a name="slide-show"></a>Slide show

适用于 Web 的 PowerPoint 在全屏窗口中播放幻灯片放映。 按下空格键可换片。 可播放幻灯片动画，但仅支持对幻灯片使用淡化和擦除切换效果。 详细了解[在浏览器与 PowerPoint 桌面应用程序中使用演示文稿的区别](https://go.microsoft.com/fwlink/?LinkId=272763)。
  
## <a name="smartart"></a>SmartArt

可以使用 PowerPoint 网页版式插入 SmartArt、切换到其他布局或配色方案、应用 SmartArt 样式和编辑文本。 详细了解[在浏览器与 PowerPoint 桌面应用程序中使用演示文稿的区别](https://go.microsoft.com/fwlink/?LinkId=272763)。
  
## <a name="table-creation-editing-and-formatting"></a>表创建、编辑和格式设置

可以在 PowerPoint 网页中创建和编辑表格，PowerPoint 网页 PowerPoint 支持大多数表格功能。 对于高级功能（如合并和拆分单元格），请使用 PowerPoint 桌面应用程序。 详细了解[在浏览器与 PowerPoint 桌面应用程序中使用演示文稿的区别](https://go.microsoft.com/fwlink/?LinkId=272763)。
  
## <a name="tell-me"></a>操作说明搜索

当你需要在 PowerPoint 网页中完成一些处理，但不知道如何完成时，可以使用"告诉我"搜索功能来快速找到要查找的内容。 "告诉我"可以理解您要完成的事项，并通过给出建议的方式帮助您更快地做到这一点。
  
## <a name="undo-and-redo"></a>撤消和恢复

在活动演示文稿的当前编辑会话期间，可撤消 (Ctrl + Z) 和重做 (Ctrl + Y) 无限次最近的操作。 如果编辑会话超时或切换到阅读视图的时间超过 30 秒，则会重置撤消历史记录。 详细了解 [PowerPoint 网页中的键盘快捷方式](https://go.microsoft.com/fwlink/?LinkId=272950)。
  
## <a name="view-and-add-comments"></a>查看和添加注释

如果文件存储在 SharePoint Online 或 OneDrive for Business 上，可以添加、编辑或删除注释。 
  
如果你的文件存储在 OneDrive 上，可以在编辑视图中添加、编辑或删除注释，并且可以在阅读视图中查看和更新注释。
  
## <a name="view-and-edit-slide-notes"></a>View and edit slide notes

借助 PowerPoint 网页，可以显示或隐藏每张幻灯片的备注。 您还可以在编辑模式下添加备注。
  
## <a name="wysiwyg-viewing"></a>WYSIWYG/所见即所得的查看

打印或显示为已完成的产品时，以与其外观十分类似的格式编辑你的演示文稿。
