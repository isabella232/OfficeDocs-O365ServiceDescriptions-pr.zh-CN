# <a name="office-365-government"></a>Office 365 政府版

> [!IMPORTANT]
> 由于冠状病毒 (COVID-19) 大流行，Microsoft Teams在线呼叫和音频/视频会议正经历着巨大的高峰。<br/>
>
>为了应对呼叫空前增加，并确保连续性和可用性，Microsoft 允许Microsoft Teams GCC音频/视频服务器利用商业数据中心和政府数据中心的处理能力。<br/>
>
>这些音频/视频服务器驻留在美国的Microsoft Azure FedRAMP 高认证边界服务器内，不存储任何客户内容。 但是，这些服务器正在处理电话和会议的音频和视频，并在此期间在我们的商业工作人员下运行。<br/>
>
>经过筛选的合格人员正在通过查看这些服务器的任何交互式登录来监视这些服务器，以获取对客户数据的潜在访问权限。 合格人员满足访问客户内容GCC要求。 有关筛选要求的详细信息，请参阅[GCC服务说明](gcc.md)。<br/>
>
>感谢您的支持，因为我们采取措施，确保我们的服务在这些特殊时期保持可用和可靠。<br/>

为了响应美国公共部门的独特和不断发展的要求，Microsoft (或Office 365 政府版) 创建了Office 365 政府版计划。 此服务说明概述了特定于Office 365 政府版美国环境的功能。 建议阅读此服务说明以及其他[Microsoft 365和Office 365服务说明](../../office-365-service-descriptions-technet-library.md)。

## <a name="how-to-use-this-service-description"></a>如何使用本服务说明

Office 365 政府版服务说明旨在用作常规Office 365服务说明的覆盖。 它定义了唯一承诺以及与 Office 365 企业版产品的差异。

## <a name="about-office-365-government-environments"></a>关于Office 365 政府版环境

Office 365 政府版计划是每月订阅，可以授权给无限数量的用户。

- **Office 365 GCC** 环境符合针对云服务的联邦要求，包括 FedRAMP High、国防联邦采购条例补充 (DFARS) ，以及 (CJI 和 FTI 数据类型) 的刑事司法和联邦税务信息系统的要求。

- **Office 365 GCC高和多维** 环境符合国防部安全要求准则、国防联邦采购条例补充 (DFARS) ，以及国际武器通信条例 (ITAR) 。

除了Office 365的功能和功能外，使用Office 365 政府版的组织还受益于Office 365 政府版特有的以下功能：

- 您组织的客户内容与 Microsoft 商业 Office 365 服务中的客户内容在逻辑上是隔离的。

- 您组织的客户内容存储在美国境内。

- 对组织的客户内容的访问权限只限于屏蔽的 Microsoft 员工。

- Office 365 政府版符合美国公共部门客户所需的认证和认证。

## <a name="customer-eligibility"></a>客户资格

Office 365 政府版可用于 (1) 美国联邦、州、地方、部落和领土政府实体， (2) 其他实体，这些实体处理受政府法规和要求约束的数据，以及Office 365 政府版的使用适合满足这些要求，但须经资格验证。 Microsoft 的资格验证包括确认处理的是受符合国际武器贸易条例 (ITAR) 约束的数据，还是受 FBI 的刑事司法信息服务 (CJIS) 政策约束的执法数据，亦或是其他受政府监管或控制的数据。 对于 ITAR 数据，资格验证可能需要提供美国国务院注册证明；对于数据处理的具体要求，可能需要提供政府机构资助证明。 Office 365 DoD 环境用于美国国防部的独占用途。

虽然资格标准在Office 365 政府版产品/服务中是一致的，但 Microsoft 将只同意 GCC 高环境的 ITAR 合同语言。

对Office 365 政府版资格有疑问的实体应咨询其帐户团队。

续订客户的Office 365 政府版合同后，需要重新验证资格。

## <a name="customer-content-located-within-the-united-states"></a>位于美国境内的客户内容

Office 365 政府版服务是从位于美国中的数据中心提供的。 客户内容以静态方式存储在仅位于美国的数据中心。这包括但不限于：

- Exchange Online邮箱内容 (电子邮件正文、日历条目和电子邮件附件的内容) 
- SharePoint联机网站内容和存储在该网站中的文件
- Microsoft Teams持久聊天线程
- Skype for Business存档的对话、上传的文档和白板会话
- Microsoft Planner计划
- 窗体数据
- PowerApps/PowerAutomate 数据

## <a name="office-365-government-and-third-party-services"></a>Office 365 政府版和第三方服务

Office 365提供了将第三方应用程序集成到 SharePoint Online 网站、Teams、Office包含在 Word、Excel、PowerPoint 等Microsoft 365 企业应用版 (中的应用程序的功能Outlook) 和Outlook Web App。 此外，Office 365 还支持与第三方服务提供商集成。 这些第三方应用程序和服务可能参与存储、传输和处理组织在 Office 365 基础结构外部的第三方系统上的客户数据，因此未涵盖在 Office 365 合规性和数据保护协议中。 我们建议您在为组织评估这些服务的相应使用情况时，查看第三方提供的隐私和合规性声明。
