# <a name="microsoft-365-education"></a>Microsoft 365 教育版

Microsoft 365有多种计划提供，以最好地满足组织的需求。 如果要了解计划与Microsoft 365计划Office 365 教育版，本文将介绍每个计划中包含的功能。
