# <a name="microsoft-365-suite-features"></a>Microsoft 365 套件功能

Microsoft Forms 允许你快速而轻松地创建自定义测验、调查、调查表、注册等。 创建测验或表单后，甚至可以在移动设备上使用任何 Web 浏览器来邀请其他人作答。 提交结果后，可以使用内置的分析来评估这些回答内容。 可以轻松地将表单数据（如测验结果）导出到Excel进行其他分析或评分。 若要了解更多信息，请参阅[什么是 Microsoft Forms？。](https://support.office.com/forms)

## <a name="windows-update-for-business-deployment-service"></a>适用于企业的 Windows 更新部署服务 

The Windows Update for Business deployment service provides control over the approval， scheduling， and protection of content delivered by Windows Update. 它旨在与现有的适用于Windows更新策略一起提供对单个更新的丰富控制。 部署服务提供的功能包括：

* 将功能更新部署安排在特定的日期开始
* 使用富表达式在几天或几周内部署阶段
* 绕过预配置的 Windows 更新 for Business 策略，以立即在整个组织中部署安全更新
* 通过针对唯一设备总体定制的部署，确保组织中硬件和软件的覆盖范围
* 自动标识并暂停对设备进行部署，这些部署可能会受保护 [保留的影响](/windows/deployment/update/safeguard-holds)

这些功能通过 Microsoft Graph API 提供，并且与 Microsoft Endpoint Manager。 若要了解更多信息，请参阅[Windows Update for Business 部署服务概述](https://aka.ms/wufb/deployment-service)。
