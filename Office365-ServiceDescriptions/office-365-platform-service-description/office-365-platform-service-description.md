# <a name="microsoft-365-and-office-365-platform-service-description"></a>Microsoft 365 和 Office 365 平台服务说明

Microsoft 365和Office 365可在各种计划中提供，以满足组织的需求。 如果要查找计划（包括Exchange Online计划）之间的差异，本文将显示每个计划中包含哪些功能。
  
Microsoft 365和Office 365将云生产力的力量传递给各种规模的企业，帮助节省时间、金钱和释放有价值的资源。 Microsoft 365和Office 365计划将熟悉的Microsoft Office桌面套件与基于云的 Microsoft 下一代通信和协作服务（包括 Microsoft Exchange Online） 版本结合在一起，Microsoft Office SharePoint Online、Office 网页版和 Microsoft Skype for Business Online， 帮助用户通过 Internet 从几乎任何地方高效工作。
  
有关为用户启用Microsoft 365和Office 365平台的订阅的详细计划信息，请参阅[完整的订阅比较表](https://go.microsoft.com/fwlink/?linkid=2139145)。 有关更详细的功能列表，请参阅[Microsoft 365和Office 365服务说明下的相关服务说明](/office365/servicedescriptions/office-365-service-descriptions-technet-library)。
  
- 有关Office 365 教育版计划的信息，请从[Office 365 教育版计划](https://aka.ms/EDU-Plan-Comparison)页开始。

- 若要了解 Office 365 美国政府版计划，请从 [Office 365 美国政府版](office-365-us-government/office-365-us-government.md)页入手。 
    
- 若要了解由世纪互联运营的 Office 365，请从[由世纪互联运营的 Office 365](office-365-operated-by-21vianet.md) 页入手。

## <a name="multi-geo-capabilities"></a>多地理位置功能 

多地理位置使单个组织能够跨多个Office 365数据中心地理位置，并使你能够按用户在所选地理位置存储Exchange Online和静态OneDrive for Business的数据。 
  
可用地理位置：亚太地区、澳大利亚、加拿大、欧盟、印度、日本、韩国、英国、美国。 
  
若要了解详细信息，请参阅[Office 365中的多地理位置简介](https://techcommunity.microsoft.com/t5/Security-Privacy-and-Compliance/Introducing-Multi-Geo-in-Office-365/ba-p/107016)。
