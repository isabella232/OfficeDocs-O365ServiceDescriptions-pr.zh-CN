# <a name="microsoft-viva-service-description"></a>Microsoft Viva 服务说明

Microsoft Viva以Teams和Microsoft 365的力量为基础，将员工体验统一到四个关键领域：参与、福利、Learning和知识，以提供一种综合体验，使人们能够成为最佳员工。 Viva 的初始模块集提供内置功能、来自 Viva 合作伙伴强大且不断增长的生态系统的集成，以及平台扩展性，使客户能够将其现有员工体验系统和工具与 Viva 集成，使其更易于员工访问和发现。

-  Viva Connections提供到数字工作场所的个性化网关，员工可以访问内部通信和公司资源（如策略和权益），并参与员工资源组等社区，所有这些都来自Microsoft Teams中的单个可自定义应用。 适用于Teams的 Connections 应用在 2021 年晚些时候推出时正式发布。

-  Viva Topics提供知识发现体验，帮助用户连接到公司中的信息和专家。 使用 AI 对客户的Microsoft 365数据进行推理，Viva Topics会自动在Microsoft 365和Teams的对话和文档中显示主题卡。 单击卡片将打开包含相关文档、对话、视频和人员的主题页面。 Viva Topics现已正式发布，作为Microsoft 365商业计划的加载项。

- **Viva Insights** 提供数据驱动的可见性，了解工作模式如何影响幸福感和工作效率以及业务性能。 个人、经理和领导者见解现已在Microsoft Teams的Viva Insights应用中提供。 Microsoft Workplace Analytics 中的Viva Insights还具有高级分析工具和模板，用于自定义、深入深入了解向经理和领导者显示的数据。 高级分析工具提供了不同的方式来分析和报告自定义分析给公司的业务主管。

    > [!NOTE]
    > Microsoft MyAnalytics和工作区分析是此新功能集的一部分，并正在更新以反映Microsoft Viva Insights品牌。

- **Viva Learning** 简化学习体验，促进成长和发展文化，将学习引入工作流程。 通过将Learning管理系统、第三方学习内容和你自己的自定义资源连接到Microsoft Teams的学习中心，在组织中发现、共享、推荐和跟踪学习。 智能工具可提供高度个性化和相关的员工学习体验，以缩小技能差距并推动个人和专业的增长。
